# SmartPlate

SmartPlate is a full-stack macro tracking web application built using Python and Flask. It allows users to track their meals, calories, macros, weight, water intake, and overall progress in a simple and accessible way without putting essential features behind a paywall.

## Features

- Log meals with automatic calorie and macro calculation  
- Store frequently used foods for faster tracking  
- Track daily calorie goals and remaining calories  
- Nutrition progress bars for protein, carbs, and fats  
- Water tracker with visual progress and goal completion  
- Weight tracking with historical trends  
- Physique progress photo uploads and side-by-side comparison  
- Charts for calories, macros, and weight over time  
- Goal planner for cutting, maintaining, or bulking  

## Tech Stack

- Python  
- Flask  
- HTML / CSS  
- JSON (for data storage)  

## How to Run Locally

1. Clone the repository:
https://github.com/Sno0pCat/SmartPlate


2. Create a virtual environment:
python -m venv venv

3. Activate It:
Windows : venv\Scripts\activate
Mac/Linux : source venv/bin/activate

4. Install dependencies:
pip install -r requirements.txt

5. Run the app:
python app.py

6. Open in browser
